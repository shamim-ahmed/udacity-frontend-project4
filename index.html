MovieBuff

An Android app for movie lovers

MovieBuff

MovieBuff is an Android app that uses the moviedb.org API to fetch various data related to popular movies. The app allows the user to sort movies based on their popularity or release date. For each movie, the synopsis, rating and trailers are shown. Moreover, the user can save the details of a particular movie for future reference.

WeatherInfo

An Android app for weather forecast

WeatherInfo

WeatherInfo is an android app that utilizes the data from the WeatherData API to show weather forecast to the user. The user is required to specify the name of his city and/or country in order to see relevant information. This app also has the capability to send forecast data to the user's Android watch.

PetRescue

A website for adopting animals

PetRescue

PetRescue is a website that allows anyone to adopt a homeless pet. A team of volunteers regularly update the website with information related to available pets. Apart from finding a new home for abandoned pets, this website also helps people find their lost pets.
